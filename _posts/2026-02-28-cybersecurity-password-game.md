# 🎮 Cybersecurity Password Strength Game

## English Version

### Welcome to the Password Security Challenge!

Test your cybersecurity knowledge with our interactive password strength game. See if you can identify weak passwords and create strong ones!

### How to Play:

1. **Analyze Given Passwords**: Rate their strength (Weak/Medium/Strong)
2. **Spot Security Issues**: Identify what makes a password vulnerable
3. **Create Strong Passwords**: Build passwords that pass security standards
4. **Earn Points**: Get rewarded for correct answers

### Password Strength Levels Explained

**WEAK Passwords (❌)**
- Password
- 123456
- admin
- qwerty
- Password1
- Your birthday or name

**MEDIUM Passwords (⚠️)**
- MyDog2024
- Secure@Home
- Coffee#Morning
- Winter2025

**STRONG Passwords (✅)**
- Tr0pic@lThunder#2024
- $uP3r$eC@re_P@ss
- QuantumLeap!892xYz
- K1ng$f1sh#Ocean98

### Game Challenge Questions

**Question 1: Is "Welcome123" a good password?**
- A) Yes, it has numbers
- B) No, it's predictable and contains common words
- C) Maybe, it depends on the system

Answer: B - It's weak because "Welcome" is a common word and "123" is a typical pattern.

**Question 2: What makes a password truly secure?**
- A) Length only
- B) Complexity + Length + Randomness
- C) Using special characters only

Answer: B - A good password needs all three elements.

**Question 3: How long should a secure password be?**
- A) 8 characters
- B) 12+ characters
- C) 20+ characters

Answer: B - Minimum 12 characters, 20+ is even better.

### Your Challenge

Try to crack these password hashes (Don't worry, these are made-up examples):

```
Hash 1: a1b2c3d4e5f6g7h8
Hash 2: password2024
Hash 3: !K@mpl3x_P@ssw0rd$
```

Which one do you think will be hardest to crack? (Answer: Hash 3 - most complex)

# 🎮 사이버보안 비밀번호 강도 게임

### 비밀번호 보안 챌린지에 오신 것을 환영합니다!

우리의 인터랙티브 비밀번호 강도 게임으로 사이버보안 지식을 테스트하세요. 약한 비밀번호를 식별하고 강한 비밀번호를 만들 수 있는지 확인하세요!

### 게임 방법:

1. **주어진 비밀번호 분석**: 강도 평가 (약함/중간/강함)
2. **보안 문제 찾기**: 비밀번호를 취약하게 만드는 요소 파악
3. **강한 비밀번호 만들기**: 보안 표준을 통과하는 비밀번호 구성
4. **포인트 획득**: 정답에 대해 보상 받기

### 게임 챌린지 질문

**질문 1: "Welcome123"은 좋은 비밀번호인가요?**
- A) 네, 숫자가 있어서
- B) 아니요, 예측 가능하고 일반적인 단어를 포함해요
- C) 경우에 따라 다릅니다

답변: B - "Welcome"은 일반적인 단어이고 "123"은 전형적인 패턴이므로 약합니다.

**질문 2: 비밀번호를 정말 안전하게 만드는 것은 무엇인가요?**
- A) 길이만
- B) 복잡성 + 길이 + 무작위성
- C) 특수 문자만 사용

답변: B - 좋은 비밀번호는 세 가지 요소가 모두 필요합니다.

**질문 3: 안전한 비밀번호의 길이는 어느 정도여야 하나요?**
- A) 8자
- B) 12자 이상
- C) 20자 이상

답변: B - 최소 12자, 20자 이상이 더 좋습니다.
